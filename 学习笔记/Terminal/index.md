# 终端命令
```
# 输出Hello World
echo "Hello World"

# 输出当前目录
pwd

#查看当前目录下所有文件信息
ls -la
drwxr-xr-x   5 zzq  staff  160  4 15  2022 工具函数 & 组件
第一列：首位:是否是文件夹，d代表文件夹、 -代表文件
后续九位为三组，代表当前用户权限，同组用户权限，其他用户权限。每组权限有3个权限位，顺序为读写执行，分别用rwx表示，当相应位置为-时，表示没有相应的权限。
第二列：硬链接数目，即有多少个文件名指向该文件。
第三列：文件所有者的用户名。
第四列：文件所属组的组名。
第五列：文件大小，单位是字节。
第六列：文件最后一次被修改的时间。
第七列：文件名。
```
